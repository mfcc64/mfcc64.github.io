Mandelbrot Set vs Julia Set

Probably, Mandelbrot set is the most popular fractal. So many videos of Mandelbrot set zoom are availabe. Julia set is not as popular as Mandelbrot set. But actually, it has close relationship with Mandelbrot set. Recently, I rendered Mandelbrot set zoom (at \( c_m \)) alongside Julia set zoom with varying control point (\(c_j\)). To make Mandelbrot set and Julia set equal in level of detail, the Julia set speed should be half of Mandelbrot set speed (represented as Mandelbrot width (\(w_m\)) and Julia width (\(w_j\))). Here are the videos (and parameters). I hope you enjoy it.

Dance of Infinity - Mandelbrot Set Zoom vs Julia Set

\[\begin{align} x_m &= -1.2522374288795870649236097248840843509256017527419668 \\ y_m &= +0.3451566295671664004327107145217337363647407791740870 \\ c_m &= x_m + i y_m \\ w_m &= 2^{-0.234375 t + 3} \\ c_j &= c_m + 0.25 w_m \sin(0.46875\pi t) e^{0.234375 \pi i t} \\ w_j &= \sqrt{8.25 w_m} \end{align}\]

Rope of Hope - Mandelbrot Set Zoom vs Julia Set

\[\begin{align} c_m &= -1.7485403467486282833966642439055485053381926480658673 \\ w_m &= 2^{-0.1171875(t-26.5) + 3} \\ c_j &= c_m + 0.25 w_m e^{0.234375 \pi i (t - 26.5)} \\ w_j &= \sqrt{8.25 w_m} \end{align}\]

The Four Craftsmen - Mandelbrot Set Zoom vs Julia Sets

\[\begin{align} c_m &= -1.7485403467486282833966642439055485053381926480658673 \\ w_m &= 2^{-0.1171875(t-26.5) + 3} \\ c_j &= c_m + 0.25 w_m e^{(0.234375 (t - 26.5) + 0.5 n + 0.25) \pi i} \hspace{0.5cm} \text{for }n = 0, 1, 2, 3\\ w_j &= \sqrt{8.25 w_m} \end{align}\]
